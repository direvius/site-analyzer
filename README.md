## Getting Started

Install the module with: `npm install site-analyzer`

```js
var site-analyzer = require('site-analyzer');
site-analyzer.awesome(); // "awesome"
```

Install with cli command

```sh
$ npm install -g site-analyzer
$ site-analyzer --help
$ site-analyzer --version
```
